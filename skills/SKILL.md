This is a navigation skill file for you. You should judge the type of question and go to the corresponding directories to find clues. Moreover, if you think the information in the corresponding directories is not enough, feel free to read more directories; that's up to you.

All paths in this document are relative paths unless otherwise specified. You must read SKILL.md in `cases` and `experience` directories first before reading content in the subdirectories of them.

We split the problems into following types:
- cpu
  - You can go to `cases/cpu` or `experience/cpu`
- memory
  - You can go to `cases/memory` or `experience/memory`
- network
  - You can go to `cases/network` or `experience/network`
- result inconsistent
  - You can go to `cases/inconsistent` or `experience/inconsistent`
- hang(sql not return)
  - You can go to `cases/hang` or `experience/hang`
- goroutine or memory leak
  - You can go to `cases/leak` or `experience/leak`
- performance
  - You can go to `cases/performance` or `experience/performance`
